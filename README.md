#### 安装教程

1. 运行环境准备mysql8+java17+npm16.9.1+redis3

2. 配置maven路径，加载依赖

3. 运行sql文件，确保application.yml的数据库名称和账号密码是数据库所在主机的账号密码

#### 使用说明

1. 登入

   账号：aaa	密码：123456

2.运行流程
